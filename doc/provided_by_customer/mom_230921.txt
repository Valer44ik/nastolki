# Экраны
## начать новую компанию
- называем, сохраняем
- выбираем тип компании (custom, lor) - от этого зависит баттлвелю меха
- выбираем сторону (всего 2) можно нечётное кол-во игроков за сторону (макс. 4)
--- выбрать formation order (lance (4 меха) или star (5 мехов) )
- за каждую сторону нужно ввести (или выбрать) имя игрока (живого)
- указывается начальный баттл вэлью (например 4000 для каждой стороны)
- указывается сумма начальные характеристик пилотов (каждому игроку одинаково)
- указывается количество пилотов (например 10) они пойдут через всю компанию
- очки пилотов распределяются между характеристиками всех пилотов игрока (мин. 2, макс. 7)
- распределяются абилки (как договорились)

## выбор компании из списка готовых (статус)
- если компания закончена, то только просмотр
- если нет, то можно продолжить/начать


## начать первый матч
- выбираем пилотов (слева)
- надо указать выбранные шасси мехов каждому пилоту (справа)
- автоматическия разбивка по группам (lance/star) - первые 4/5 - первая группа
внизу показать общий баттлвелю всех выбранных пилотов и мехов (не должен привышать согласованное значение)
если есть остаток от тотал БВ, то он переходит на следующий матч
- указать кол-во глобальных и локальных задачь для каждой стороны

## закончить матч
- глобальная задача выполнена или нет (для каждой стороны)
- локальные задачи (неограниченное количество) тоже выполнена/нет
-

## начать новый матч
- можно поменять шасси (требует времени)
- между боями проходит 1 неделя игрового времени (это равно смена модели, излечение 1ог ранения пилота)


# Админстративная консоль (Mech Database)
- создаём шасси (рэйвн, масса, класс назначается автоматом в зависимости от массы)
- создаём модель меха (шасси, имя модели и 2 баттлвелю)
--- у меха есть владелец (игрок). Владелец может меняться.
- создаём пилотов (имя, фамилия, позывной)
---- пилот принадлежит конкертному игроку


# характеристики пилотов (может быть дробное значение)
- piloting 
- shooting
--- макс. характеристка 1го пилота не может быть больше 7 и меньше 2
хар-ки меняются между матчами


# Абилки пилотов
- зависят от характеристик (открываются по мере улучшения)
- название
- список бонусов
- список штрафов
- некоторы абилки недоступны кланам (бывают ограничения, обсудить отдельно)
- абилки "включаются" автоматически по мере прокачки пилота
